linux fdisk 工具

時間： 2019-11-09   |   分類： Centos 7   | 字數： 112 字 | 閱讀： 1分鐘 | 閱讀次數：

fdisk

用來建立硬碟的分割表

我使用的 fdisk 版本為 2.33.1

有支援 GPT、MBR 分割表

 

顯示目前有哪些硬碟
fdisk -l



把硬碟初始化，因為我的硬碟有使用過
所以先做 partition pable 重建 GPT ，把所有資料清乾淨

fdisk /dev/sda
o   create a new empty DOS partition table
g   create a new empty GPT partition table
w   write table to disk and exit
 

fdisk /dev/sdc
g
w

fdisk /dev/sdd
g
w
